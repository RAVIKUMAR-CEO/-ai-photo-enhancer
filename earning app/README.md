# 🌟 AI Photo Enhancer

A free, AI-powered web app that enhances your photos with professional-grade filters and quality analysis.

## ✨ Features

- **AI Photo Analysis** - Get quality score & enhancement suggestions
- **Professional Filters** - Auto-enhance, sharpen, brighten, contrast, warm/cool tones, denoise, vintage
- **Before/After Comparison** - See the difference instantly
- **Payment Integration** - Monetize downloads via Razorpay
- **100% Free Tier** - No backend needed, uses Google AI Studio

## 🚀 Deployment

Fully deployed on GitHub Pages - just open `index.html` in your browser!

**Live URL:** https://RAVIKUMAR-CEO.github.io/ai-photo-enhancer

## 🔧 Setup (Local Development)

1. Clone this repo
2. Open `ai-photo-enhancer-paid.html` in your browser
3. Upload a photo and start enhancing!

## 💳 Payment Setup

To enable downloads:
1. Get Razorpay account at [razorpay.com](https://razorpay.com)
2. Replace `RAZORPAY_KEY` in the HTML with your key

## 🤖 AI Engine

- **Provider:** Google AI Studio (Gemini 2.0 Flash)
- **Free Tier:** 60 requests/minute, 1M tokens/day
- **Get API Key:** https://aistudio.google.com/apikey
